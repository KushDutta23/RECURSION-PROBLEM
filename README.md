📘 Recursion Practice in C++

This project demonstrates various recursion techniques in C++, including printing sequences, calculating sums, factorial, reversing arrays, and checking for palindrome strings.

🚀 Features

The program includes the following recursive functions:

🔢 Print numbers from 1 to N
🔄 Print numbers from N to 1
🔁 Backtracking (forward and reverse)
➕ Sum of first N numbers (two approaches)
✖️ Factorial of a number
🔃 Reverse an array
🔍 Check if a string is a palindrome
🧠 Concepts Used
Recursion
Backtracking
Arrays
Strings
Function calls and stack behavior
📂 Functions Overview
1. printname(int i, int n)

Prints numbers from 1 to N

2. printreverse(int i, int n)

Prints numbers from N to 1

3. printbacktracking(int i, int n)

Prints numbers from 1 to N using backtracking

4. printbtreverse(int i, int n)

Prints numbers from N to 1 using backtracking

5. printsum(int n)

Returns the sum of first N numbers

6. printsumP(int n, int sum)

Prints sum using parameterized recursion

7. fact(int n)

Returns factorial of N

8. reverseArray(int i, int arr[], int n)

Reverses the array using recursion

9. stringPalindrome(int i, string &s)

Checks whether a string is a palindrome
